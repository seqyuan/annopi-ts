## 前置条件

- **Node.js ≥ 18**
- 本地执行需要 **ata**（[`../ata`](https://github.com/seqyuan/ata)）
- 集群投递需要 **annotask**（[`../annotask`](https://github.com/seqyuan/annotask)）— 若所有任务都是 `executor: local` 则可省略

## 安装

```bash
npm install -g @seqyuan/annopi
```

或使用 pnpm：

```bash
pnpm add -g @seqyuan/annopi
```

验证：

```bash
annopi --help
```

## 运行示例流程

仓库包含从 Python annopi 复制的 fixtures：

```bash
annopi conf \
  -p tests/fixtures/annopi/pipeline.yml \
  -c tests/fixtures/annopi/project.yml \
  -o /tmp/annopi-out
```

生成结果：

```
/tmp/annopi-out/
├── shell/
│   ├── 1-0-cellranger.sh
│   └── 2-0-qc.sh
└── tasks.yml
```

查看输出：

```bash
cat /tmp/annopi-out/tasks.yml
```

执行流程：

```bash
annopi run -o /tmp/annopi-out
```

已完成的任务会在 `shell/` 下写入 `.sign` 文件。再次执行 `annopi run` 会跳过已完成任务。

## 开发

若要参与 annopi-ts 开发，可克隆 monorepo：

```bash
git clone https://github.com/seqyuan/annopi-ts.git
cd annopi-ts
pnpm install
pnpm build
pnpm test
pnpm typecheck
```

链接本地 CLI：

```bash
cd packages/cli
npm link
cd ../..
annopi conf -p ... -c ... -o ...
```
