## Prerequisites

- **Node.js ≥ 18**
- **ata** for local task execution ([`../ata`](https://github.com/seqyuan/ata))
- **annotask** for qsubsge cluster submission ([`../annotask`](https://github.com/seqyuan/annotask)) — optional if all tasks use `executor: local`

## Install

```bash
npm install -g @seqyuan/annopi
```

Or with pnpm:

```bash
pnpm add -g @seqyuan/annopi
```

Verify:

```bash
annopi --help
```

## Run the Example Pipeline

The repo includes fixtures copied from Python annopi:

```bash
annopi conf \
  -p tests/fixtures/annopi/pipeline.yml \
  -c tests/fixtures/annopi/project.yml \
  -o /tmp/annopi-out
```

This generates:

```
/tmp/annopi-out/
├── shell/
│   ├── 1-0-cellranger.sh
│   └── 2-0-qc.sh
└── tasks.yml
```

Inspect the output:

```bash
cat /tmp/annopi-out/tasks.yml
```

Then execute:

```bash
annopi run -o /tmp/annopi-out
```

Completed tasks write a `.sign` file under `shell/`. Re-running `annopi run` skips done tasks.

## Development

Clone the monorepo if you want to work on annopi-ts itself:

```bash
git clone https://github.com/seqyuan/annopi-ts.git
cd annopi-ts
pnpm install
pnpm build
pnpm test
pnpm typecheck
```

Link the local CLI:

```bash
cd packages/cli
npm link
cd ../..
annopi conf -p ... -c ... -o ...
```

## Documentation Site

```bash
pnpm site:dev
```

Open [http://localhost:3000](http://localhost:3000) for the English home page, or
[http://localhost:3000/zh](http://localhost:3000/zh) for Chinese.
